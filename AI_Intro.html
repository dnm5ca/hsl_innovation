Intro to Artificial Intelligence

What is artificial Intelligence (AI)

Artificial intelligence (AI) encompasses a broad field that is primarily interested in training intelligent machines which can help solve problems. From that broad categorization, AI can be broken down into more specific layers:

  • Artificial intelligence is the broadest category and encompasses all fields involved with creating intelligent machines.
  • Machine learning involves pattern recognition in structured data which is often stored in a tabular format and often employs statistical methods. In fact, linear regression is a supervised machine learning technique.
  • Deep learning is a subset of machine learning that uses artificial neural networks, a computational method inspired by the human brain. Because it uses more complex methods, deep learning is better equipped to handle unstructured data types like images, text, or audio.
  • Large language models are specifically designed for text. AI chatbots like ChatGPT are based on a LLM which has been trained on a specific set of data. LLMs are also considered generative AI since they are used to produce new content.

    • General Guidelines for AI Use

    If you’re new to working with AI, there are a few foundational ideas that you should remember. The UVA Arts & Sciences Learning Design and Technology group has provided a few foundational ideas on how to utilize generative AI (GenAI) that are also provided below:

    Foundational Ideas

    • There always needs to be a “human in the loop” to verify output from AI.
    • Generative AI is not a web search. It will generate a new response every time.
    • You are conversing with AI, not just asking it questions. How you write your prompt matters
    • We are all participating in the discovery of AIs.

    It is especially important to remember that a LLM’s primary goal is to generate human-like text, not true statements. The following flow chart from a UNESCO report indicates that the first question you should ask when using ChatGPT or a similar AI is “Does it matter if the output is true?” When using generative AI, you must have the expertise to validate the output and accept responsibility for any decisions or final products.

    Data Privacy and Ethical Concerns

    Not only do you need to cautious of the results produced by generative AI, but you should also be careful about what information you upload as well. Depending on the particular model you’re using, it may store information from your sessions and use it for training purposes. Even if the company is not using that data for training purposes, if previous sessions are being stored so that you can revisit them, there is still the potential for a data breach. The current recommendation from UVA is to not enter sensitive, personal, or proprietary information into any Gen-AI system and to use UVA-licensed AI tools whenever possible.

    Prompting GenAI

    There are many different prompting frameworks, but there are a few key components that are normally included.

    • Instruction: A specific task for the model to perform
    • Context: Additional details or information you can provide the model for better responses
      • Role: Defining a role or persona the AI should model its response after (medical librarian, statistician, manuscript reviewer)
    • Input: Any input data (text or data to be analyzed)
    • Output format: How the output should be formatted (formal text, bullet points, tables, etc. )

    Tips and Tricks

    • Use delimiters (like the triple quotes above) to indicate distinct parts of the input
      • In ChatGPT, you can tab down with shift+enter
    • Specify the steps required to complete a task
    • Provide examples for the AI to model its response after
    • Give the model time to “think”
      • “Let’s work this out in a step by step way to be suer we have the right answer”
    • Remember this is an iterative process: use follow-up prompts
    • Use LLMs to help you study
      • Generate flash cards
      • Ask a LLM to generate case studies and then practice diagnosing them
      • Generate quizzes and tests

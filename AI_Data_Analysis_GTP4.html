ChatGPT 4: Data Analyst GPT

Full access to ChatGPT 4 requires a paid membership. This version offers additional capabilities, such as the ability to upload external files. For security and privacy, only publicly available data should be uploaded. ChatGPT 4 also includes access to specialized GPTs like DALL-E and Data Analyst.

The Data Analyst GPT can write and run Python code directly in your browser, eliminating the need to run code locally. It can assist with all stages of data analysis, including:

  • Data cleaning and preparation
  • Data exploration including summary statistics and visualization
  • Formal hypotheses testing and interpretation of results

    • While these features are very helpful, it’s important to verify the results. In this tutorial, we'll learn how to use the Data Analyst GPT, including how to view and verify the code produced by ChatGPT.

    Tutorial: Data Cleaning and Exploration

    Scenario: We have been given a dataset on the annual number of object launched into space by different countries. Our goal is to perform some exploratory data analysis and generate initial plots. We’ve decided to use the Data Analyst GPT for this task.

    First, ensure that either GPT-4 or GPT-4-turbo is selected. You can verify this by checking the model selector in your ChatGPT session. Then to start using the Data Analyst GPT, upload your data file by either dragging and dropping it into the chat or by clicking the paperclip icon on the left side of the prompt box and navigating to the file. Recently, the option to connect to cloud storage services like Google Drive or OneDrive has been added. However, due to privacy concerns, we recommend only uploading publicly available data.

    Once you have attached the file, you can either upload it without a text prompt or provide additional information. We’ll start by asking ChatGPT 4 to provide a summary of the dataset. ChatGPT 4 will use Python to read the file and generate summary information. While the Python script is running, ChatGPT 4 will display "analyzing".

Of course, we would still want to verify the results.

    Key takeaways

    ChatGPT and other GenAI models can be helpful in your data analysis, but there are certain things you must remember when using it:

  • Only upload publicly available data
  • Not all tasks should be delegated to ChatGPT
  • Never accept results without verifying them
  • This is an iterative process and you will most likely have to refine results
